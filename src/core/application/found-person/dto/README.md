# dto (found-person)

Contratos de datos **del backend** (snake_case). NO se usan en la UI: solo
viajan por HTTP y los mappers los traducen a/desde el modelo de dominio.

- `found-person-request.dto.ts` — cuerpo del `POST /api/report`.
- `found-person-response.dto.ts` — item de `GET /api/found-persons`.
